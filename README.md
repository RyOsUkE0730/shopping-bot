# Shopping Bot

## 概要
LINE Bot APIとRakuten APIを活用し、<br>
「LINEアカウント内で商品名を送ると、楽天市場内のランキング上位3つの商品を
リプライで返してくれる」<br>というチャットbotを作成しました。
